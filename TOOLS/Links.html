Mobile 3D Game Development: From Start to Finish

Art Tools:

To create art assets using the same techniques as shown in the book, you will need download the Mascot Capsule™ M3G Version 4 Toolkit and the appropriate plug-in for 3D applicaiton of choice. All of these tools are available from the same link from HI Corporation.

  1. Mascot Capsule™ M3G Toolkit (Mascot Capsule V4)
  2. Mascot Capsule™ 3DS Max™ exporter
  3. Mascot Capsule™ Maya™ exporter
  4. Mascot Capsule™ LightWave3D™ exporter
  5. Mascot Captuse™ SOFTIMAGE|XSI™ exporter
